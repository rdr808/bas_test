# BAS test task

## Описание

Это приложение на Node.js сокращает URL-адреса и перенаправляет пользователей по коротким ссылкам.

## Технические детали

- **POST /shorten**: создает сокращенный URL
- **GET /:shortcode**: перенаправляет на оригинальный URL
